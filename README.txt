زاد المتون
==========

النسخة الحالية تحتوي على:
- الرئيسية
- المتون
- القرآن
- الدروس

إضافة متن جديد:
1) ارفع ملف PDF إلى:
   assets/pdf

2) افتح:
   assets/data/mutoon.json

3) أضف سطرًا جديدًا مثل هذا:
{
  "title": "الأربعون النووية",
  "category": "حديث",
  "description": "أحاديث جامعة.",
  "file": "assets/pdf/al-arbaeen.pdf",
  "keywords": "الأربعون النووية حديث"
}

إضافة درس جديد:
1) افتح:
   assets/data/lessons.json

2) أضف سطرًا جديدًا مثل هذا:
{
  "title": "شرح الأصول الثلاثة",
  "teacher": "قناة علمية",
  "type": "يوتيوب",
  "url": "https://www.youtube.com/",
  "description": "سلسلة شرح مختصرة."
}

مهم:
- لا يوجد أي وصف يشير إلى أنه قالب ذكاء اصطناعي.
- جدول المتون جاهز: اسم المتن + التصنيف + زر التحميل.
- تم تكبير صفحات القرآن وتحسين التنقل بينها.

- تم إضافة التصفح بالأجزاء أو بالسور.


مهم:
- إذا فتحت الملفات مباشرة من الجهاز بدون رفعها للاستضافة، قد تمنع بعض المتصفحات قراءة JSON.
- في هذه النسخة تم تحويل بيانات المتون والدروس أيضاً إلى ملفات JS لتعمل محلياً وعلى الاستضافة.

- تم تصحيح أسماء ووصف الأجزاء في قسم القرآن.
